# Format numbers without the leading zero

Format a numeric vector to character, stripping the leading zero from
values whose absolute value is less than 1. `0.123 -> ".123"`,
`-0.045 -> "-.045"`, `1.234 -> "1.234"`.

Used as a vignette / output style helper: funfield's vignettes default
to no leading zeros on decimals (aggressive: applies to all decimals,
not just bounded-by-1 quantities).

## Usage

``` r
f0(x, digits = 3, keep_sign = FALSE)
```

## Arguments

- x:

  Numeric vector (or anything coercible to numeric). `NA` passes through
  as `NA_character_`.

- digits:

  Number of decimal places. Default 3.

- keep_sign:

  Logical; if `TRUE`, prepend an explicit `"+"` for non-negative
  non-zero values. Default `FALSE`.

## Value

A character vector the same length as `x`.

## Examples

``` r
f0(c(0.123, -0.045, 1.234, NA))         # ".123", "-.045", "1.234", NA
#> [1] ".123"  "-.045" "1.234" NA     
f0(c(0.1, -0.001), digits = 2)          # ".10", "-.00"
#> [1] ".10"  "-.00"
f0(0.5, keep_sign = TRUE)               # "+.500"
#> [1] "+.500"
```
