# Expected Effects of Variable-of-Interest

Give the average effects associated with this type of situation
specified as the situational variable of interest (s_voi)

Note: this function is currently defunct; it doesn't handle 'i' feature
appropriately I believe same functionality exists through combination of
`siWeightedData` and `ecov_long` and `ecov`

## Usage

``` r
expEffects.voi(ESJT, voiset, voi)
```

## Arguments

- ESJT:

  Expected effect data

- voiset:

  situational variable of interest

- voi:

  name of the variable of interest

## Value

expected effects for situational VOI
